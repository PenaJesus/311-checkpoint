Recipe Guide

GET /recipes

This is a GET method. It allows the user to see all recipes

GET /recipes/:id

This is also a GET method.
It allows the user to see a certain recipe by entering a specific ID.

POST /recipes

This is a POST method.
It allows the user to add a new recipe.

PUT /recipes/:id

This is a PUT method.
It allows the user to update a specific recipe with the specified ID.

DELETE /recipes/:id

This is a DELETE Method.
It will allow the user to delete a recipe with the specified ID
